# Guía de Clase 1 -- Introducción al Machine Learning

## Objetivo

Al finalizar la clase los alumnos deberían: - Comprender qué es Machine
Learning. - Conocer el flujo general de un proyecto de ML. - Tener el
entorno funcionando. - Ejecutar su primer modelo de Machine Learning. -
Entender qué veremos durante el curso.
